# TaskFlow

TaskFlow is a responsive full-stack task manager with basic authentication, CRUD operations, and a polished dashboard UI.

## Setup instructions

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm start
   ```
3. Open http://localhost:3000
4. Sign in with one of the sample accounts:
   - demo / demo123
   - admin / admin123
